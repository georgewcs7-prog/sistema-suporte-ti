# Sistema de Suporte Técnico

Este projeto simula um sistema de atendimento de suporte técnico, identificando problemas comuns e sugerindo soluções automaticamente.

## Funcionalidades
- Identificação de problemas (internet, lentidão, erros)
- Sugestão automática de soluções
- Simulação de abertura de chamado

## Tecnologias utilizadas
- Python
- Lógica de programação

## Como executar
1. Instale o Python
2. Baixe o projeto
3. Execute no terminal:
   python suporte_ti.py

## Objetivo
Projeto desenvolvido para prática de lógica de programação e simulação de rotinas de suporte técnico.
